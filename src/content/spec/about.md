# PAISL 소개

PAISL은 **피지컬 인공지능 보안 연구반**입니다. 임베디드 시스템과 인공지능이 만나는 지점을 보안의 관점에서 탐구하며, 실제 시스템을 이해하고 설계하고 검증할 수 있는 인재를 양성하는 것을 목표로 합니다.

## 우리가 다루는 주제

- 임베디드 시스템과 마이크로컨트롤러
- 운영체제, 컴퓨터구조, 디지털 논리회로
- 머신러닝, 딥러닝, LLM, 온디바이스 AI
- 피지컬 AI 환경에서의 보안 연구와 개발

## 설립 목적

PAISL은 컴퓨터과학, 전산학, 인공지능, 보안 지식을 함께 갖춘 인재를 키우기 위해 만들어졌습니다.

- 피지컬 인공지능 시대에 필요한 융합형 역량을 체계적으로 학습합니다.
- 아직 충분히 활성화되지 않은 피지컬 인공지능 보안 분야를 연구하고 개발합니다.
- 기초 이론부터 실제 프로젝트까지 이어지는 학습 흐름을 만듭니다.

## 운영 방식

- **주 2회 정규 활동**으로 기초 이론과 실습을 병행합니다.
- 상반기에는 C, 자료구조, 알고리즘, 컴퓨터구조, 운영체제, 디지털 논리회로를 다집니다.
- 여름 이후에는 임베디드 시스템과 AI, 온디바이스 AI를 연결해 프로젝트 단계로 넘어갑니다.
- 하반기에는 팀별 프로젝트를 통해 연구 결과를 구현하고 발표합니다.
